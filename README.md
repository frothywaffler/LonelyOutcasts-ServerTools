# About Me

Hey, I'm **Colton Eller**, better known in Path of Titans as **FrothyWaffler**.

I'm the owner and operator of **Lonely Outcasts**, a Path of Titans community server built around events, player interaction, and trying to create a fun place for people to play.

I did not start out knowing all of this. A lot of what I have learned came from jumping in, breaking things, fixing them, asking questions, and figuring it out one step at a time. Running Lonely Outcasts has pushed me into learning Linux, Bash scripting, server automation, troubleshooting, event planning, and community management.

Over time, I have built custom server tools, automated maintenance systems, teleport systems, timeout systems, event areas, and quality-of-life features to make the server better for players and easier to manage for staff.

# Modding Goals

My goal is to become a Path of Titans mod creator and eventually build a custom map specifically for the Lonely Outcasts community.

I have spent a lot of time using Creator Mode and community-created mods from other Path of Titans creators. Seeing what people have been able to build has made me want to learn the modding pipeline myself and eventually create my own content.

I am still early in the modding journey, but I have looked around Unreal Engine and have some outside familiarity with it. I am also actively looking into Blender so I can start learning asset creation. I tend to learn fast when I get locked into a project, and this is something I genuinely want to get good at.

My plan is to start small with simple server-focused mods, event props, and Creator Mode assets that can be used on Lonely Outcasts. While I am learning and getting comfortable with the tools, anything I create will stay exclusive to my server so I can test it properly and avoid releasing broken or unfinished content to the public.

Once I feel confident in the quality of my work, I would like to branch out and release polished mods, assets, and eventually maps for the wider Path of Titans community.

# Long-Term Vision

The ultimate goal is to create a custom map designed specifically for **Lonely Outcasts**.

I want that map to support large community events, unique environments, server-specific gameplay, and the kind of player experience that fits the community we have been building.
