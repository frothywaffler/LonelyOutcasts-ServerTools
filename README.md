# Modding Goals

My goal is to become a Path of Titans mod creator and eventually design a custom map built specifically for the Lonely Outcasts community.

Through operating my server and building event infrastructure, I have gained experience working with Creator Mode and community-created mods developed by other Path of Titans creators. Seeing what talented creators have built for the game has inspired me to learn the modding pipeline myself and eventually contribute content of my own.

While I am still early in the modding journey, I already have some familiarity with Unreal Engine and am actively learning Blender. I enjoy learning technical skills and tend to become heavily invested in projects that interest me, allowing me to learn new tools and workflows quickly.

My plan is to start small by creating server-focused assets, event props, and Creator Mode content that can be used and tested within Lonely Outcasts.

Initially, any mods or maps I create will remain exclusive to Lonely Outcasts. My focus is on learning the tools properly, refining my work, and producing quality content rather than rushing releases.

As my skills and confidence grow, I hope to begin releasing polished mods, assets, and eventually maps that can be enjoyed by the wider Path of Titans community.

### Planned Learning Path

- Continue Learning Unreal Engine
- Learn Blender & 3D Asset Creation
- Obtain Path of Titans Modding Access
- Create Simple Server-Focused Mods
- Develop Custom Assets & Event Props
- Thoroughly Test Content on Lonely Outcasts
- Release Public Mods for the Community
- Design & Build a Custom Map for Lonely Outcasts

# Long-Term Vision

The ultimate goal is to create a custom map designed specifically for Lonely Outcasts, supporting large-scale community events, unique environments, and gameplay experiences tailored to the server's player base.

Once I reach that point, I hope to contribute additional mods, assets, maps, and experiences to the wider Path of Titans community as well.
