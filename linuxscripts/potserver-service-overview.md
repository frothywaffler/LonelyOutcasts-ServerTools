# Path of Titans Dedicated Server Service

This project demonstrates the Linux systemd service configuration used to manage and maintain the Lonely Outcasts Path of Titans dedicated server.

The service is responsible for launching the server, monitoring its status, automatically recovering from failures, and ensuring reliable long-term operation.

---

# Features

- Automatic startup during system boot
- Automatic restart after crashes or unexpected shutdowns
- Dedicated service account execution
- Controlled working environment
- Process supervision through systemd
- Integration with automated maintenance systems
- Improved server reliability and uptime

---

# Technologies Used

- Ubuntu Linux
- systemd
- Bash
- Path of Titans Dedicated Server

---

# Purpose

Managing the game server through systemd provides a reliable and professional method of server administration.

By utilizing service management and automated recovery mechanisms, the server can recover from crashes, restart automatically after maintenance, and remain available with minimal manual intervention.
