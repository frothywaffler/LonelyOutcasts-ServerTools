# Path of Titans Dedicated Server Service

This project uses Linux systemd service management to keep the dedicated Path of Titans server running automatically.

## Features

- Automatic startup on boot
- Automatic restart after crashes
- Dedicated service account execution
- Defined working directory
- Process supervision through systemd
- Integration with automated maintenance scripts

## Technologies Used

- Ubuntu Linux
- systemd
- Bash
- Path of Titans Dedicated Server

## Purpose

Using systemd allows the game server to automatically recover from crashes, start during system boot, and integrate with scheduled maintenance workflows.
