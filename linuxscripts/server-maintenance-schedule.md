# Automated Server Maintenance Schedule

This project documents the automated maintenance systems used to manage and protect the Lonely Outcasts Path of Titans dedicated server.

Routine administrative tasks are handled through scheduled Linux cron jobs, reducing manual workload and improving server reliability.

---

# Automated Tasks

## Creator Mode Backup

Creates automated backups of Creator Mode data to protect community-built structures and event infrastructure.

## Full Server Backup

Generates complete server backups to protect configuration files, scripts, and operational data.

## Pre-Update Backup

Creates a recovery point before major updates or maintenance procedures.

## Automated Daily Restart

Performs scheduled server restarts to maintain stability, reload services, and ensure consistent performance.

---

# Technologies Used

- Linux Cron

- Path of Titans Dedicated Server

---

# Purpose

The maintenance schedule provides automated protection against data loss while minimizing administrative overhead.

By combining routine backups with scheduled maintenance procedures, the server remains stable, recoverable, and easier to manage over long periods of operation.
