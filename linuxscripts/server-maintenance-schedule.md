# Automated Server Maintenance Schedule

This server uses Linux cron scheduling to automate routine maintenance tasks.

## Scheduled Tasks

- Daily Creator Mode backup
- Daily full server backup
- Pre-update backup
- Automated daily restart

## Technologies Used

- Cron
- Bash
- Linux Administration
- Path of Titans Dedicated Server

## Purpose

Automated scheduling reduces downtime, protects server data, improves reliability, and minimizes manual administrative work.
