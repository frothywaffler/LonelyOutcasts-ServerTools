# Creator Mode Builds

This folder contains examples of Creator Mode work from the Lonely Outcasts Path of Titans community server.

## Examples Included

- Fight Night Arena
- Arena Entrance
- Spectator Viewing Area
- Staff/Admin Observation Area
- Food & Water Area
- Player Staging Area
- Community Gathering Spaces

## Skills Demonstrated

- Event planning
- Player flow design
- Arena construction
- Spectator management
- Creator Mode building
- Community event organization

## Purpose

These builds were created to support community events hosted on the Lonely Outcasts server. They demonstrate practical Creator Mode experience and serve as examples of my interest in future Path of Titans mod creation and eventual map development.
